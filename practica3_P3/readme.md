url: Dirección del recurso consultado.

metodo: Método HTTP utilizado (GET, POST, DELETE, etc.).

status: Código de estado numérico devuelto por la API (ej. 200, 404, 500).

categoria: Rango del código HTTP (2xx, 3xx, 4xx, 5xx).

tipo: Nombre descriptivo del tipo de respuesta o error.

accion: Recomendación o acción sugerida para corregir o procesar la petición.

exitoso: Valor booleano (true si el status está entre 200 y 299; false en caso contrario).
